xFun API

## Description

xFunserver.net API for authorization, stats, staffs, etc.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
